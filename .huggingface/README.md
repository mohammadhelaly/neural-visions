# NeuralVisions

A web app for answering open-ended questions about images, developed using Python, Flask and React.js.

This Space deploys the production runtime image from the GitHub repository [here](https://github.com/mohammadhelaly/neural-visions).

The source of truth for the project lives in GitHub. This Space is intended to be updated through GitHub Actions.
